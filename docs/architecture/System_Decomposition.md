# System Decomposition
## Modules
 - User Management
 - Schedule Management
 - Assignment Tracking
 - Calender and Events
 - Notification and Reminders
 - Study Planner
## Component Responsibilities
 - User Management : Handle student registration, login, logout, and profile management.
 - Schedule Management: Store and manage class timetables, module time slots, and detect scheduling conflicts.
 - Assignment Tracking: CRUD operations for assignments, progress tracking by section, and deadline management.
 - Calendar & Events: Aggregate classes, assignments, and personal events into a unified daily and weekly view.
 - Notifications & Reminders: Trigger deadline alerts and class reminders based on student preferences.
- Study Planner: Create and manage study sessions, suggest available time slots, and track study streaks.
