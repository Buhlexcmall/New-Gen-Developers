# Features List

---

## Must Have Features
*(Core features required for the app to function — highest priority)*

- **User Authentication** — Students can register, log in, and manage their account securely
- **Class Schedule Management** — Students can add, view, edit, and delete class schedules in a weekly calendar view
- **Assignment Tracking** — Students can create, update, and delete assignments with due dates and status labels (To Do / In Progress / Done)
- **Deadline Notifications and Reminders** — Automated alerts remind students of upcoming assignments and class times
- **Dashboard Overview** — A home screen summarising today's classes, upcoming deadlines, and pending tasks

## Should Have Features
*(Important features that add significant value — implement after Must Haves)*

- **Study Session Planner** — Students can schedule and track dedicated study sessions linked to specific subjects
- **Task Priority Levels** — Students can mark tasks as High, Medium, or Low priority
- **Progress Tracking** — Visual indicators (progress bars, completion percentages) showing how much of a module's work is done
- **Search and Filter** — Students can search assignments by subject or filter by status and due date

## Could Have Features
*(Nice-to-have enhancements — implement if time allows)*

- **Dark Mode** — Alternative colour theme for low-light environments
- **Google Calendar Sync** — Export class schedules and deadlines to Google Calendar
- **Peer Study Groups** — Students can create or join study groups within the app
- **Academic Performance Notes** — A notes section per subject for tracking grades and feedback
